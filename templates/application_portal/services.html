    Services

    The project focused on service delivery to the system users on basis of information,tools and capabilities of the system. The workflow implemented in this project included the change of user. the pilot LADM provides various services including change of user evaluation application, application status and parcel information search among others.The services available in the pilot system are subject to improvement based on the upscale strategy to be rolled out with time.

    The services implemented in this system help the users navigate and accomplish tasks related to land transactions in Kenya. The services assume the following (as per this pilot system):


    • The land owner/user has already secured the required documents for the respective land transaction e.g. The physical plan , comments and the receipt of payment.
    • The ministry/land management body has details about the land parcels and the owners.

    In summary the services available at the moment include the following:

    • Land transaction application service
    • Application status portal
    • Mapping portal (information on landuse, landcover e.t.c)
    • Administration dashboard
    • Single account user service

    The listed services are the so-far achieved services and more are expected to be incoorporated with time.
